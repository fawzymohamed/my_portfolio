# Portfolio Website Overall Plan

## Objectives
- Build a bilingual (EN/AR) portfolio website based on the Nuxt UI Portfolio template.
- Showcase full-stack expertise with security-first positioning.
- Feature the Stock Management System as the top project.
- Deploy on Vercel with production-ready SEO and performance.

## Scope
- Use Nuxt 4 + Nuxt UI (free) and follow the existing template structure.
- Populate content from the provided CV.
- Implement i18n with RTL support for Arabic.
- Add or remove sections based on relevance (testimonials, blog, speaking, FAQ).

## Primary Audience
- Hiring managers, engineering leaders, and enterprise stakeholders.

## Information Architecture
- Home (hero, about, experience, projects, contact)
- Projects (all projects)
- About (long-form bio)
- Optional: Blog, Speaking, FAQ (if kept)

## Key Content Priorities
- Strong hero statement with value proposition.
- Evidence-backed impact metrics.
- Clear contact and social links.

## Technical Direction
- Nuxt 4 + Nuxt UI components.
- Content driven by Nuxt Content YAML.
- i18n for /en and /ar routes with RTL for Arabic.
- Static assets in public/ with optimized images.
- Vercel deployment with environment variables and build settings.

## Deliverables
- Updated content files with EN/AR copy.
- Updated components/layouts for language switch and RTL support.
- Project entries with Stock Management System first.
- Vercel deployment configuration and verification.

## Assumptions
- Template repo is present locally.
- Links for LinkedIn/GitHub will be provided or can be added later.
- Project screenshots may be placeholders initially.

## Success Criteria
- Responsive and accessible UI in both languages.
- Consistent design with template.
- Clear navigation and contact CTA.
- Deployed and live on Vercel.
